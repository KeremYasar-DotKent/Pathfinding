Sena Karaca

Game Description:
CodeLogic: Pathfinding is a puzzle-based learning game developed in Unity that aims to teach students the fundamentals of logic and algorithmic thinking.

The player’s objective is to guide a character through a grid-based level to reach a target destination. Instead of direct control, the player must "program" the character’s movements by placing command blocks (such as Move Forward, Turn Left, and Turn Right) in a sequential order. Once the "Execute" button is pressed, the character follows the commands.

Educational Elements:
Algorithmic Thinking: Players learn how to break down a problem into a sequence of steps.

Problem Solving: Players must analyze the environment and obstacles to find the most efficient route.

Sequential Logic: The game emphasizes that the order of instructions is critical to achieving the desired outcome.
